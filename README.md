# 3rd-party-tools

## flomo-extension
flomo-extension 是一款适用于 Chrome 内核的浏览器插件，专门为 flomo 开发。

不同于其他插件需要借助于 flomo 的 Pro 会员权限，flomo-extension 只需要使用时登录网页版本的 flomo 账户即可使用。
